# 🗳️ Premium Blockchain E-Voting System

A state-of-the-art, secure, and transparent e-voting application built with **Ethereum Blockchain**, **React**, **Node.js**, and **MongoDB**.

## ✨ Features
- **Immutable Voting:** All votes are recorded on the blockchain using Solidity smart contracts.
- **Vibrant Dashboard:** Premium UI with glassmorphism, animations, and real-time stats.
- **MetaMask Integration:** Secure wallet connection for voter identity and transaction signing.
- **Admin Portal:** Secure election creation and candidate management.
- **Live Results:** Animated charts showing real-time voting progress.

## 🚀 Tech Stack
- **Frontend:** React, Tailwind CSS v4, Framer Motion, Ethers.js
- **Backend:** Node.js, Express, MongoDB, JWT Auth
- **Blockchain:** Solidity, Hardhat, Ethereum Local Node

## 🛠️ Setup Instructions

### 1. Prerequisites
- Install [Node.js](https://nodejs.org/)
- Install [MongoDB](https://www.mongodb.com/try/download/community)
- Install [MetaMask](https://metamask.io/) browser extension

### 2. Run the Application
1. Double-click the `Run_EVoting_System.bat` file on your Desktop.
   - This will start the Backend, Frontend, and the local Blockchain node automatically.
2. Open your browser to `http://localhost:5173`.
3. Connect your MetaMask wallet to the **Localhost 8545** network.
